## 🚀 실행 방법

### 1. 백엔드 서버 실행
```bash
cd PythonProject
# envs/.local.env 파일 생성 후
docker compose up -d --build
```

### 2. Flutter 실행
```bash
cd flutter_application_1
flutter pub get
flutter run
```

## 📡 API 연동

- Base URL: `http://localhost/api`
- 인증: Bearer JWT Token
- 백엔드: FastAPI + MySQL + Redis
